# Due to all the skids this repository will not recieve any more updates

This project aims to build a functional GTA Online server. The main goal is to **get contributions from the community** and **implement all endpoints** so the server can run properly.

> **Note:** This server currently supports **PS3 only**. **Xbox and Xbox 360 support are not implemented**, as testing hardware is unavailable.

## Running Locally

To test the server locally, GTA Online needs to redirect certain Rockstar endpoints to your server. This can be done via **DNS configuration** or by editing your **hosts file**.  

Domains to redirect:

- `prod.gtao.ca`
- `prod.cloud.gtao.ca`
- `cloud.gtao.ca`
- `tunables.gtao.ca`
- `ros.gtao.ca`
- `prod.ros.gtao.ca`
- `accounts-prod.ros.gtao.ca`
- `feed-gta5-prod.ros.gtao.ca`
- `prod.telemetry.ros.gtao.ca`
- `ps-gta5-prod.ros.gtao.ca`
- `mm-gta5-prod.ros.gtao.ca`
- `ugc-gta5-prod.ros.gtao.ca`
- `prs-gta5-prod.ros.gtao.ca`
- `inbox-gta5-prod.ros.gtao.ca`
- `crews-gta5-prod.ros.gtao.ca`
- `cs-gta5-prod.ros.gtao.ca`
- `services.ros.gtao.ca`
